Windows NT

a new development team within Microsoft began work on a revamped version of IBM and Microsoft's OS/2 operating system known as "NT OS/2". NT OS/2 …it Used the same program manager as in the Windows 3.1, designed to be multi-user operating system, more secured O.S. as you can manage who can sees your files, came out with performance manager to monitor your device’s use rate, manage who can access the system, as you can run as administrator to get all the system’s rigts to administrate it, also included network management features, to manage groups of servers at the same point of management as it contained server manager program….The first release of the resulting operating system, Windows NT 3.1 in July 1993, with versions for desktop workstations and servers. Windows NT 3.5 was released in September 1994, focusing on performance improvements and support for Novell's NetWare, and was followed up by Windows NT 3.51 in May 1995, which included additional improvements and support for the PowerPC architecture. Windows NT 4.0 was released in June 1996, introducing the redesigned interface of Windows 95 to the NT series. On February 17, 2000, Microsoft released Windows 2000, a successor to NT 4.0. The Windows NT name was dropped at this point in order to put a greater focus on the Windows brand.

Windows XP

The next major version of Windows NT, Windows XP, was released on October 25, 2001. The introduction of Windows XP aimed to unify the consumer-oriented Windows 9x series with the architecture introduced by Windows NT, a change which Microsoft promised would provide better performance over its DOS-based predecessors. Windows XP would also introduce a redesigned user interface including an updated Start menu and a "task-oriented" Windows Explorer, streamlined multimedia features as the new Media Center with a support for DVD playback, TV tuner cards, DVR functionality, and remote controls and networking features, Internet Explorer 6, integration with Microsoft's .NET Passport services, modes to help provide compatibility with software designed for previous versions of Windows, and Remote Assistance functionality.. Mainstream support for Windows XP ended on April 14, 2009. Extended support ended on April 8, 2014.

Windows Vista

was released on November 30, 2006, included a redesigned shell and user interface , with a particular focus on security features. It was available in a number of different editions, was a faster boot.

Windows 7

Was released in July, 2009. Unlike its old man, Windows Vista, Windows 7 was more usable, including upgrades to the Windows old features, with the goal of being compatible with applications and hardware with which Windows Vista was already compatible.with multi-touch support, a redesigned Windows shell with an updated taskbar, a home networking system called HomeGroup, and performance improvements.

Windows 8

was released in October, 2012. Came with upgrades in the user interface especially for the touch-based devices such as tablets. These upgrades include the Start screen, which uses large tiles that are more convenient for touch interactions and allow for the display of continually updated information, and a new class of apps which are designed primarily for use on touch-based devices, so you can say that windows 8 didn’t serve computers as much as it served touch-based devices.

Windows 8.1

Was released in Oct.2013,came with increasing integration with cloud services, Microsoft's OneDrive, the Windows Store service

Windows 10

was released in July, 2015, and addresses shortcomings in the user interface first introduced with Windows 8. Changes on PC include the return of the Start Menu, a virtual desktop system, and the ability to run Windows Store apps within windows on the desktop rather than in full-screen mode.
